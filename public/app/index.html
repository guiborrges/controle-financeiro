Widget para iPhone

Carregando status...
Como usar: 1. Instale o app Scriptable no iPhone (App Store, gratuito)
2. Abra o Scriptable e toque em + para criar novo script
3. Cole o código copiado e toque em Done
4. Na tela inicial do iPhone, pressione e segure → toque em +
5. Busque por Scriptable e adicione o widget médio
6. Toque no widget → selecione o script criado

O widget usa um token seguro e individual. Nunca compartilhe esse token. Você pode revogar e gerar um novo a qualquer momento.

Primeiros passos

Por onde eu começo?

Comece pelo Controle dos Meses: registre renda, despesas e metas financeiras. Depois use Patrimônio para acompanhar onde o dinheiro guardado está.

Qual a lógica do sistema?

O produto foi dividido em 4 papéis: Dashboard para visão geral, Controle dos Meses para operar o mês, Patrimônio para o dinheiro acumulado e Histórico para comparar evolução.

Preciso preencher tudo?

Não. O núcleo do uso diário é simples: renda, despesas, gastos diários e, se quiser, metas. O resto aprofunda a leitura sem ser obrigatório.

Controle dos Meses

Como usar os blocos do mês?

Cada bloco segue a mesma lógica: veja o total, use + Adicionar e clique nas linhas para editar. Você também pode arrastar os blocos para a ordem que preferir.

Para que servem as metas financeiras?

Elas representam dinheiro planejado para separar, não gasto realizado. Você pode escolher a conta de destino e depois enviar esse valor para o Patrimônio.

Como os gastos diários ajudam?

Os gastos diários mostram gasto, meta e saldo por categoria. O sistema tenta sugerir títulos já usados, traz a data atual por padrão e coloca o lançamento novo no topo para evitar duplicidade.

Patrimônio

O que entra em Patrimônio?

Contas ou reservas onde você guarda dinheiro, como caixa, caixinha, CDI, poupança, conta corrente ou cripto. O saldo muda só por movimentações manuais.

Como mover dinheiro entre contas?

Selecione uma conta à esquerda e use os botões de +, − ou transferência. Transferências mudam os saldos das contas, mas não alteram o patrimônio total.

Como as metas se conectam ao patrimônio?

No Controle dos Meses, cada meta pode apontar para uma conta patrimonial. Quando você enviar a meta, o sistema cria a movimentação correspondente naquela conta.

Dashboard e Histórico

Qual a diferença entre Dashboard e Histórico?

O Dashboard resume o período selecionado com indicadores, tabelas e gráficos. O Histórico é para comparar meses e enxergar tendência com mais foco.

Posso reorganizar o Dashboard?

Sim. As caixas do Dashboard continuam independentes e redimensionáveis. Se algo sair do lugar, você pode resetar o layout em Preferências.

Quando usar filtros?

Use filtros para aprofundar. A ideia do topo das páginas é leitura rápida; os filtros refinam a análise quando você precisar.

Segurança e backups

Como funciona o backup?

Você pode criar backup manualmente pelo botão da lateral. O sistema também gera backup automático em regras internas do projeto e mantém histórico por usuário.

Exportar é a mesma coisa que backup?

Não exatamente. Exportar gera um arquivo para levar seus dados. Backup é a camada de proteção e restauração do sistema.

Mexer em preferências apaga meus dados?

Não. As preferências afetam layout, visual e comportamentos de interface. Seus dados financeiros continuam separados dessas configurações.
