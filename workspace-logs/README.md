# Workspace Logs

Esta pasta guarda registros de apoio do trabalho feito no projeto.

Arquivos:

- `assistant-log.md`: respostas e resumos relevantes do assistente.
- `command-log.md`: logs resumidos de comandos executados a pedido do usuario.

Observacoes:

- Os registros passam a valer a partir desta configuracao.
- Nem toda resposta do chat sera copiada literalmente; quando fizer mais sentido, sera salvo um resumo fiel e objetivo.
- Logs muito grandes podem ser resumidos para facilitar leitura.
